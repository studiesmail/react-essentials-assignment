# Task Manager App

A simple Task Manager built using **React, Context API, and useReducer** to manage global state efficiently.

## Features

* Add, edit, and delete tasks
* Mark tasks as complete
* View total & completed tasks
* Responsive UI with basic animations

## Tech Stack

* React
* Context API
* useReducer
* CSS

## Project Structure
```
src/
 ├── context/
 │    ├── TaskContext.js
 │    ├── taskReducer.js
 ├── components/
 │    ├── TaskInput.js
 │    ├── TaskList.js
 │    ├── TaskItem.js
 │    ├── TaskSummary.js
 ├── App.js
 ├── index.js
 ├── styles.css
```
## Run Locally
Installation & Setup
```
git clone https://github.com/your-username/task-manager.git
cd task-manager
npm install
npm start
```
http://localhost:3000
